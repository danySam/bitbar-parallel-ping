# bitbar-parallel-ping

An [xbar/bitbar](https://github.com/matryer/xbar) plugin to show latency (ping) with Google and Cloudflare DNS servers (1.1.1.1 and 8.8.8.8)

## Usage
Move the *.cgo file to your xbar plugin folder
